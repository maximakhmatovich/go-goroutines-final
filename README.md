# Yandex Practicum Go Goroutines Sprint 9 Final

Этот проект является итогом моего обучения ***горутинам***, **конкурентному программированию** и работе с **параллельными вычислениями в Go**.

Программа демонстрирует параллельную обработку данных для поиска максимального значения в большом массиве случайных чисел.

## Описание


- В один поток (последовательно)
- В несколько потоков (используя горутины для параллельной обработки)

## Описание

Программа генерирует 100 миллионов случайных целых чисел и находит среди них максимальное значение двумя способами. 

Реализованы два подхода:

1. **Однопоточный поиск** — обычный проход по массиву
2. **Параллельный поиск** — разделение массива на части и обработка с помощью **goroutines**

Это позволяет сравнить производительность последовательного и параллельного выполнения.

## Функционал

- **generateRandomElements** — генерация большого массива случайных чисел
- **maximum** — поиск максимального элемента в слайсе
- **maxChunks** — параллельный поиск максимального значения с использованием goroutines
- **main** — сравнение времени выполнения однопоточной и многопоточной версии

## Как работает параллельный поиск

1. Массив делится на **CHUNKS частей**
2. Для каждой части запускается **goroutine**
3. Каждая goroutine находит максимум своего куска
4. После завершения всех goroutines:
   - собираются промежуточные результаты
   - находится **финальный максимум**

Синхронизация выполняется через **sync.WaitGroup**.
